# Package index

## Table constructors

Build publication-ready tables from data frames.

- [`tbl_descriptive()`](https://clerkr.circadia-lab.uk/reference/tbl_descriptive.md)
  : Descriptive summary table with group comparisons (Table 1)

## Render

Render a clerk_tbl to gt, HTML, or LaTeX.

- [`render()`](https://clerkr.circadia-lab.uk/reference/render.md) :
  Render a clerk_tbl to its target output format
- [`render_gt()`](https://clerkr.circadia-lab.uk/reference/render_gt.md)
  : Render a clerk_tbl as a gt table (Word / PDF)
- [`render_reactable()`](https://clerkr.circadia-lab.uk/reference/render_reactable.md)
  : Render a clerk_tbl as an interactive HTML table
- [`render_latex()`](https://clerkr.circadia-lab.uk/reference/render_latex.md)
  : Render a clerk_tbl as a LaTeX table

## Theme & palette

clerkR visual identity — colours, scales, and gt theme.

- [`clerk_theme()`](https://clerkr.circadia-lab.uk/reference/clerk_theme.md)
  : Apply the clerkR gt theme to a gt table
- [`clerk_palette()`](https://clerkr.circadia-lab.uk/reference/clerk_palette.md)
  : clerkR colour palette
- [`clerk_colour()`](https://clerkr.circadia-lab.uk/reference/clerk_colour.md)
  : Access a single clerkR colour by name
- [`clerk_diverging()`](https://clerkr.circadia-lab.uk/reference/clerk_diverging.md)
  : Build a clerkR diverging colour scale
- [`clerk_sequential()`](https://clerkr.circadia-lab.uk/reference/clerk_sequential.md)
  : Build a clerkR sequential colour scale
